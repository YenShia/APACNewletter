# APAC Newsletter

A sleek, interactive dashboard and newsletter platform built for the APAC (Asia-Pacific) region — designed to deliver key insights, updates, and performance metrics in a visually engaging format.

## Overview

APAC Newsletter is a single-page HTML application featuring a modern dark-themed UI with dynamic visual effects. It serves as a centralized hub for sharing regional updates, product highlights, and business intelligence across APAC markets.

## Features

- Dark-themed responsive dashboard interface
- Animated visual effects (floating glow spheres, grain overlays)
- Clean typography with Poppins / system font stack
- Fully self-contained — no build tools required
- Mobile-friendly layout

## Tech Stack

- **HTML5** — semantic markup
- **CSS3** — custom animations, gradients, radial effects
- **JavaScript** — interactivity and dynamic content
- **Tailwind CSS** (CDN) — utility-first styling
- **Google Fonts** — custom typography
- **Font Awesome** — iconography

## Getting Started

1. Clone the repo
```bash
   git clone https://github.com/YenShia/APACNewletter.git
```

2. Open `index.html` in your browser — no server needed.

3. Or visit the live site: [yenshia.github.io/APACNewletter](https://yenshia.github.io/APACNewletter/)
